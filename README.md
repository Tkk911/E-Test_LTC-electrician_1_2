# E-Test_LTC-electrician_1_2
แบบทดสอบมาตรฐานวิชาการติดตั้งไฟฟ้าภายในอาคาร ระดับ 1   ชุดที่ 2 จำนวน 100 ข้อ
 หมายเหตุ ไม่ใช่ข้อสอบจริงจากกรมพัฒนาฝีมือแรงงาน เป็นเพียงแนวทางในการเตรียมตัวสอบภาคความรู้(ทฤษฎี)เท่านั้น 
 ผู้เข้ารับการทดสอบ จะต้องทำข้อสอบ จำนวน 36 ข้อขึ้นไป จากจำนวน 60 ข้อ จึงมีสิทธิ์เข้ารับการสอบภาคปฎิบัติ ต่อไป
